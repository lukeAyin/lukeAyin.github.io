# 633 平方数之和

> 给定一个非负整数 c ，你要判断是否存在两个整数 a 和 b，使得 a2 + b2 = c 。
>
>  
>
> 示例 1：
>
> 输入：c = 5
> 输出：true
> 解释：1 * 1 + 2 * 2 = 5
> 示例 2：
>
> 输入：c = 3
> 输出：false
> 示例 3：
>
> 输入：c = 4
> 输出：true
> 示例 4：
>
> 输入：c = 2
> 输出：true
> 示例 5：
>
> 输入：c = 1
> 输出：true
>
>
> 提示：
>
> 0 <= c <= 231 - 1
> 链接：https://leetcode-cn.com/problems/sum-of-square-numbers

## 双指针

~~~c++
#define int64 long long
class Solution {
public:
    bool judgeSquareSum(int c) {
        int r=(int)sqrt(c);
        int l=0;
        while(l<=r){
            int64 t=l*l;
            t+=r*r;
            if(t>c){
                r--;
            }else if(t==c){
                return true;
            }else{
                l++;
            }
        }
        return false;
    }
};
~~~
